QR-Code-Generator
=================

You can generate QR Code in just seconds and save it in the form of .jpg, .png and .gif formats too!

Requirements:
PHP Server (v5.2.* and above)
Winzip

How to operate:
1. Unzip the package
2. Change the background (If wanted)
3. Host the unziped package on your hosting
4. Enjoy!
